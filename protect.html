Engagement Overview

One Platform,
Five Initiatives

PROTECT (Prosecutor Technology for Case Tracking) is the primary case management system used by District Attorneys across Wisconsin. Over 18 months, I led the UI/UX design for five interconnected projects — each solving a distinct problem while contributing to one cohesive, standards-compliant system.

01 / Brand
PROTECT Logo & Identity
Created the visual identity — logo, color palette, and type system — that anchors all five products under a single recognizable brand.
View case study →
02 / Desktop Feature
Details Pane Redesign
Redesigned the in-app Details Pane to eliminate tab-switching friction, reduce lookup time, and align with modern UX standards for legal professionals.
View case study →
03 / Web Tool
eFiling Site
Designed a web-based eFiling interface that pulls nightly data from all county databases, enabling fast cross-county case and person lookups.
View case study →
04 / Web Application
XCQ Inter-County Query
Redesigned the legacy XCQ web app with a modern design system, improved data grid, and streamlined search flows for inter-county case research.
View case study →
05 / Desktop App
Desktop Application
Improved the core PROTECT Windows desktop application — the primary daily workflow tool for all Wisconsin District Attorneys managing active case files.
