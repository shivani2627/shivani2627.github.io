The Challenge
One bank.
Scattered data.
🗂️
The Problem
Citibank employees had to jump across multiple internal tools, spreadsheets, and email threads to track project statuses, module KPIs, and ownership across different business units. There was no single source of truth — leading to delays, duplicated effort, and accountability gaps.
🎯
The Solution
Design and deliver an internal Spotlight Dashboard — a centralized portal where every Citibank module appears as a KPI card with live data, project ownership, and status tracking. Built on Citi's own design system to ensure brand consistency and faster adoption by internal teams.

Key Features
What we built
📊
KPI Cards
Every Citibank business module gets its own KPI card — showing live metrics, ownership, and project status at a glance. Configurable by team leads and refreshed in real time.
Data VizReal-timeConfigurable
🌗
Dark & Light Mode
Full dark/light theme toggle built for accessibility. Both themes tested to WCAG AA contrast standards, using Citi's approved color tokens — not custom one-offs.
AccessibilityWCAG AACiti Tokens
🏛️
Citi Design System
Every component — buttons, cards, tables, forms — uses Citi's own internal design system. This meant faster developer handoff, guaranteed brand consistency, and zero custom component debt.
Component LibraryBrand Compliance
👤
Ownership Tracking
Every module and project is tied to an owner — with avatars, names, and status. Reduces the classic "who owns this?" back-and-forth and makes accountability visual.
Project MgmtTransparency
🔍
Unified Search
Employees can search across all modules, projects, and owners from a single search bar — eliminating the need to navigate between separate tools or ask colleagues where data lives.
SearchDiscovery
🔔
Role-based Access
Different views for different roles — employees see their modules, managers see cross-unit dashboards, admins see the full picture. Access designed to reduce noise and surface relevance.
PermissionsRole UX
Design System
Built on Citi's foundation
Rather than building a custom design system from scratch, the Spotlight Dashboard was designed entirely within Citi's existing internal design system. This created immediate familiarity for internal users and significantly reduced the engineering effort required for implementation.
